## Summary

- Dashboard first paint now comes from one bundled payload instead of separate chart and widget requests.
- Search previews skip the extra count query when a section does not overflow, which cut `search_animals` from `3` queries to `2` and `search_all` from `13` queries to `7`.
- Core list screens now derive totals from the current page window when possible and only fall back to `COUNT(*)` on overflow.
- The `After Dashboard Cache` section captures the warm repeated-dashboard path, where the bootstrap route dropped to `1` query. The final `After Optimization` table is still useful for the cold optimized request path after the short cache TTL expires.
